Exercise 2: File Creation and Processing

  1. Open the terminal and create a new text file called sample.txt using nano. Enter the text below.

  2. This is the first line of the file.
    The quick brown cat jumps over the lazy dog.
    Linux is a powerful operating system.
    Linux allows you to build custom workflows.
    Pipes are used to chain commands together.
    Nano is a simple text editor.
    Nano is often used for quick edits.
    This is the last line of the file.

  3. Save and exit the file.

  4. Display the contents of the file using cat.

  5. Show the contents of the file along with line numbers using the -n option with cat.

  6. Find lines containing the word "Linux" using grep.

  7. Find lines ignoring case sensitivity by using the -i option with grep.

  8. Display the matching lines along with their line numbers using -n with grep.

  9. Display lines that do not contain the word "Linux" using grep -v.

  10. Count the number of lines in the file using wc -l.

  11. Count the number of words in the file using wc -w.

  12. Count the number of characters in the file using wc -m.

  13. Count the number of bytes in the file using wc -c.

  14. Display the first 3 lines of the file using head -n.

  15. Display the last 2 lines of the file using tail -n.

  16. Filter lines containing "Linux" and redirect the output to a file called linux_lines.txt.

  17. Append the result of searching for "Nano" to the same file, linux_lines.txt.

  18. Intentionally search for a non-existent file and redirect the error output to error_log.txt.

  19. Combine cat, grep, and wc to find how many lines contain the word "the" (case-insensitive).

  20. Use tee to send the output of the filtered lines with "Linux" to both the terminal and a file called result.txt.

Solutions:

Create the file with nano:

nano sample.txt

Enter the text in consecutive lines:

Save and exit the file by pressing:

`Ctrl + O`, then `Enter`, and finally `Ctrl + X`.

Display the file content:

cat sample.txt

Show file content with line numbers:

 cat -n sample.txt

Find lines containing "Linux":

 grep "Linux" sample.txt

Find lines ignoring case sensitivity:

 grep -i "linux" sample.txt

Display line numbers for matching lines:

 grep -n "Linux" sample.txt

Display lines that do not contain "Linux":

 grep -v "Linux" sample.txt

Count lines in the file:

wc -l sample.txt

Count words in the file:

wc -w sample.txt

Count characters in the file:

wc -m sample.txt

Count bytes in the file:

wc -c sample.txt

Display the first 3 lines:

head -n 3 sample.txt

Display the last 2 lines:

tail -n 2 sample.txt

Redirect lines containing "Linux" to a file:

grep "Linux" sample.txt > linux_lines.txt

Append the result of "Nano" search to the same file:

grep "Nano" sample.txt >> linux_lines.txt

Redirect an error message to error_log.txt:

grep "anything" nonexistent.txt 2> error_log.txt

Count the number of lines with "the":

cat sample.txt | grep -i "the" | wc -l

Send output to both a file and the terminal with tee:

grep "Linux" sample.txt | tee result.txt
