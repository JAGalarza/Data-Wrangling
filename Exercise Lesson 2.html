Exercise: Advanced Text Manipulation with cat, cut, sort, paste, and Redirection:

Use nano to create a file with the text below. Name the file sample.txt

This is the first line of the file.
The quick brown cat jumps over the lazy dog.
Linux v2.4 is a powerful operating system.
Linux V.3.6 allows you to build custom workflows.
Pipes are used to chain commands together.
Nano is a simple text editor.
Nano is often used for quick edits.
This is the last line of the file.
  1. Display the file content: Use `cat to display the entire content of sample.txt.
  2. Display the file content with line numbers: Show each line of sample.txt prefixed with its line number.
  3. Extract and sort the first word of each line: Extract the first word from each line and display them in alphabetical order.
  4. Sort the lines in reverse alphabetical order: Sort the lines of sample.txt in reverse order.
  5. Display only the second column (second word) of each line: Use cut to extract and display the second word of each line.
  6. Combine the extracted first word with the full original line: Extract the first word of each line, and paste it next to the corresponding original line.
  7. Redirect lines that contain the word "Linux" to a new file: Redirect lines that mention "Linux" to a new file called linux_lines.txt.
  8. Display lines containing the word "Nano" and redirect to stderr: Sort the lines containing "Nano" and display them to stderr.
  9. Create a summary file of the first word and full line: Create a new file with two columns: the first word of each line and the full line.
  10. Display only unique lines: Filter out and display only unique lines, ignoring duplicates. lace all spaces in the file with tabs:** Use sed to replace all spaces with tabs in the file.
  11. Count the number of characters in the file: Use wc -m or awk to count the total number of characters in the file.

Solution Section:

1 Display the file content:

cat sample.txt

This command simply displays the entire contents of sample.txt.

2 Display the file content with line numbers:

cat -n sample.txt

Using the -n option, cat shows each line of sample.txt with line numbers prefixed.

3 Extract and sort the first word of each line:

cut -d ' ' -f 1 sample.txt | sort

The cut command extracts the first word of each line, and sort orders them alphabetically.

4 Sort the lines in reverse alphabetical order:

sort -r sample.txt

The -r option of sort displays the lines in reverse alphabetical order.

5 Display only the second column (second word) of each line:

cut -d ' ' -f 2 sample.txt

The cut command with the -f 2 flag extracts the second word of each line.

6 Combine the extracted first word with the full original line:

paste <(cut -d ' ' -f 1 sample.txt) <(cat sample.txt)

This uses process substitution to combine the first word and the full line.

7 Redirect lines that contain the word "Linux" to a new file:

grep "Linux" sample.txt > linux_lines.txt

This command uses grep to search for lines with "Linux" and redirects the output to linux_lines.txt.

8 Display lines containing the word "Nano" and redirect to stderr:

grep "Nano" sample.txt | sort 1>&2

This command filters lines containing "Nano", `sorts them, and redirects the output to stderr.

9 Create a summary file of the first word and full line:

paste &lt;(cut -d ' ' -f 1 sample.txt) <(cat sample.txt) &gt; summary.txt

This command creates a two-column summary of the first word and the full line and writes it to summary.txt.

10 Display only unique lines:

sort sample.txt | uniq

This sort | uniq combination filters out duplicate lines.

11 Count the number of characters in the file:

wc -m sample.txt

The wc -m command counts and displays the total number of characters in sample.txt.
